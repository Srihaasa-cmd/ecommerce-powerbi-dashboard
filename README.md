# ecommerce-powerbi-dashboard
## Overview
Built an interactive Power BI dashboard analysing 1,000 e-commerce 
transactions across 5 product categories and 4 regions to track 
business performance and identify growth opportunities.

## Problem Statement
The business had raw sales data but no easy way to monitor 
performance, compare categories or filter results by region 
and payment method in real time.

## Dashboard Features
- 4 KPI cards — Total Revenue, Total Orders, Avg Order Value, Return Rate
- Monthly revenue trend line chart showing seasonal patterns
- Category revenue bar chart identifying top performing products
- Payment method pie chart showing customer preferences
- Region revenue chart comparing geographical performance
- Interactive slicers to filter entire dashboard by category and region

## Key Findings
- Electronics drives the highest revenue despite not being 
  the highest volume category
- Revenue peaks in specific months — suggesting seasonal demand patterns
- UPI and Credit Card are the most preferred payment methods
- Certain regions consistently outperform others in revenue

## Tools Used
- Power BI Desktop — dashboard building and DAX measures
- Excel — initial data exploration
- DAX — custom measures for KPIs

## Files
- `ECommerce_Dashboard.pbix` — Power BI dashboard file
- `dashboard_screenshot.png` — preview of the dashboard
- `ecommerce_data.csv` — dataset used

## How to Open
1. Download Power BI Desktop from powerbi.microsoft.com
2. Open `ECommerce_Dashboard.pbix`
3. Explore the dashboard using slicers and filters

## Dataset
Simulated e-commerce dataset — 1,000 orders, 286 unique customers, 2024
